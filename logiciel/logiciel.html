Fonctionnalité

Ce logiciel dispose de 3 grandes fonctionnalités :
  • D’afficher en temps réel les données émises par Arduino Due qui dispose d’une plateforme accélérométrique et gyroscopique. On peut visualiser ces données sous la forme :
    • D’une courbe représentant les 3 axes en temps réel.
    • Des données bruts avec des afficheurs 7 segments.
  • De stocker ces données sous la forme d’un fichier CSV pour pouvoir traiter ces données sur Matlab/Ocatve/Scilab.
  • De filtrer les donnés reçues à l’aide d’un filtre FIR (filtre à réponse impulsionnel finie).
  • De commander l’envoi des données de l’Arduino Due.

Commande

Due Command

  • Open ouvre le port série pour communiquer avec l’Arduino. (Si l’ouverture échoue il ne se passe rien).
  • Close ferme le port série pour communiquer avec l’Arduino.
  • Start on démarre l’émission périodique des données de l’Arduino.
  • Avec Stop et Clear on met en pause.
  • Clear fait repartir la courbe de 0.

Figure 4 Commande port série

Le port série

Il faut choisir le port correpondant au port au est connecter l’Arduino DUE

Le filtrage

Vous pouvez charger un filtre de type FIR qui filtrera en temps réel les donnés reçues.
Pour charge un nouveau filtre cliquer sur New Filter
Figure 6 filter

Exportation d’un fichier CSV

  • Pour enregistrer les données cochez la case ON/OFF.
  • Pour effacer le fichier actuel cliquer sur new acquisition.
  • Export va vous demander ou sauvegarder votre fichier CSV.
